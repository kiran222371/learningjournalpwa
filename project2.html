Project 2: JavaScript Interaction

This project introduced the use of JavaScript to add interactivity and dynamic functionality to my website. I implemented a hamburger menu, theme switcher, and live date/time display. Through this, I learned how JavaScript interacts with the DOM using methods like getElementById() and querySelector(), and how event listeners trigger visual changes. Debugging and testing in the browser console helped me gain confidence in using developer tools effectively.
