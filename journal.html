Weekly Learning Journal

Each week I’ll reflect on my learning journey for the Mobile Application Development course.

Week 2 Reflection

1️⃣ How did you approach mobile-first design?

I began with the smallest screen layout, writing base CSS without any media queries. Once it looked good on mobile, I added breakpoints for tablets and desktops. This ensured that the design grew naturally instead of shrinking down.

2️⃣ What was the most useful HTML or CSS concept you applied this week?

Flexbox and Grid were the most helpful. Flexbox helped align the header items, and CSS Grid made it easy to arrange project cards responsively.

3️⃣ What part of HTML or CSS did you find most challenging or confusing?

Media queries were initially confusing—understanding which styles override others. I practiced by resizing the browser and using the Inspector to watch how CSS changed.
